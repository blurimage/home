ΟΔΗΓΕΙΕΣ ΕΓΚΑΤΑΣΤΑΣΗΣ:

Για την εκτέλεση του προγράμματος είναι αναγκαία η εγκατάσταση τών module pygame και pillow.

Καί στις δύο περιπτώσεις πρώτα ανοίγουμε την Γραμμή Εντολών.

Για την εγκατάσταση ,τής pygame γράφουμε :
pip install pygame 

Για την εγκατάσταση  της Pillow γράφουμε:
pip install  Pillow

Να σημειωθεί οτι το πρόγραμμα που πρέπει να τρέξει ειναι το My_Quiz.py
